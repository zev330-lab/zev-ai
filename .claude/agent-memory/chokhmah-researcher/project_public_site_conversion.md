Research completed 2026-03-22. Expansion doc at `.tola/research/public-site-conversion-expansion.md`.

Critical findings: zero client testimonials, no headshot/video, only one conversion path (12-step form), no social proof near CTAs, "we" language masking solo practitioner, no middle-of-funnel content, no FAQ page, no privacy policy, and the impressive TOLA admin pipeline is invisible to prospects.

**Why:** The public site needs to generate consulting leads. Currently it has sophisticated design and architecture but lacks the conversion fundamentals that move visitors to action — trust signals, multiple entry points, objection handling, and human connection.

**How to apply:** Implementation should prioritize (1) adding a human face/headshot, (2) client testimonials near CTAs, (3) a low-friction conversion path (calendar booking or short form), and (4) post-submission pipeline visibility before feature additions.
